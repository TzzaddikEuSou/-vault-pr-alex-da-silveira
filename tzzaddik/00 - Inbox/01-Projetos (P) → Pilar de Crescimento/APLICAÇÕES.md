# 🧠 Wireframe Cognitivo — App de Superaprendizagem

## 🎯 Objetivo Central

Fixação profunda de conteúdo usando:

- Repetição espaçada
    
- Active recall
    
- Micro-reforços triplos (1:3 adaptado para retenção)
    
- Gamificação leve e estratégica
    
- Foco absoluto durante estudo
    

---

# 1️⃣ ONBOARDING — Calibragem do Hardware Cognitivo

### Objetivo:

Personalizar o algoritmo antes do primeiro estudo.

### Wireframe:

------------------------------------------------  
| Logo Minimal                                 |  
|                                              |  
| Qual seu objetivo principal?                 |  
| ( ) Provas                                   |  
| ( ) Sermões                                  |  
| ( ) Idiomas                                  |  
| ( ) Concursos                                |  
|                                              |  
| Quantos minutos por dia? [ 20 ]              |  
|                                              |  
| Nível de retenção desejado?                  |  
| ( ) Básico                                   |  
| ( ) Avançado                                 |  
| ( ) Domínio                                  |  
|                                              |  
| [ INICIAR JORNADA ]                          |

🔎 Boa prática:

- Nada de animações pesadas aqui.
    
- UX objetiva.

# 3️⃣ SESSÃO DE ESTUDO — Zona de Foco Absoluto

Esta é a sala sagrada da retenção.

Sem navbar.  
Sem distrações.  
Sem vidro obsidiano exagerado.

### Wireframe:

------------------------------------------------  
| Pergunta                                     |  
|                                              |  
| O que significa justificação em Romanos 5?  |  
|                                              |  
|                                              |  
| [ MOSTRAR RESPOSTA ]                         |  
------------------------------------------------

Após clicar:

------------------------------------------------  
| Resposta                                     |  
|                                              |  
| Declaração jurídica de justiça...           |  
|                                              |  
| Como foi sua lembrança?                     |  
|                                              |  
| [ Difícil ] [ Médio ] [ Fácil ]             |  
------------------------------------------------

💡 Aqui entra o algoritmo SRS.

Sem animações contínuas.

# 📐 Princípios Cognitivos Aplicados

1. Tela única por foco
    
2. Um CTA dominante por tela
    
3. Redução de carga cognitiva
    
4. Feedback imediato
    
5. Progressão visível
    
6. Recompensa moderada
    
7. Dados > Decoração
    

---

# 🎨 Onde Entra o Midnight Luxe

Use estética forte apenas em:

- Dashboard
    
- Progresso
    
- Landing institucional
    

Nunca na tela de estudo.

Durante recall, o cérebro precisa de silêncio visual.

---

# 🔥 Fluxo Ideal do Usuário

Notificação  
↓  
Dashboard  
↓  
Sessão (Recall ativo)  
↓  
Feedback  
↓  
XP  
↓  
Volta ao Dashboard

Ciclo curto.  
Recompensa rápida.  
Retenção acumulativa.
